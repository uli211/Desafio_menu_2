Completar la implementación en C\# las clases indicadas a continuación, con sus responsabilidades y colaboraciones también indicadas a continuación. Cada clase debe ser agregada por un estudiante diferente y deben trabajar en ramas diferentes.\hypertarget{README.md_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{Pasos}{Pasos}}\label{README.md_autotoc_md5}

\begin{DoxyEnumerate}
\item Uno de los estudiantes del equipo crea un repositorio privado a partir de este repositorio plantilla haciendo clic en el botón {\ttfamily Use this template} y luego en el comando {\ttfamily Create new repository}.
\item El estudiante que creó el repositorio agrega a los demás integrantes del equipo haciendo clic en {\ttfamily Settings}, {\ttfamily Collaborators and Teams}, {\ttfamily Add people} para cada uno de los miembros del equipo, usando sus nombres de usuario.
\item Todos clonan el repositorio creado a sus equipos.
\item Todos crean una rama con el nombre que quieran, la única condición es que deben ser diferentes para cada miembro del equipo.
\item Distribuyan las siguientes clases entre los miembros del equipo. Cada miembro implementa una clase diferente.
\item El código no va a funcionar hasta que integren el trabajo en la rama {\ttfamily main}, por lo que deberán unir los cambios con frecuencia.
\end{DoxyEnumerate}
